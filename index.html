Available Commands

            

                

                    
Movement

                    
    
                        
  • FORWARD n or FD n - Move forward n pixels
    • 
                        
  • BACKWARD n or BK n - Move backward n pixels
    • 
                        
  • LEFT n or LT n - Turn left n degrees
    • 
                        
  • RIGHT n or RT n - Turn right n degrees
    • 
                        
  • SETXY x y - Move to position (x, y)
    • 
                        
  • SETX n - Set X position to n
    • 
                        
  • SETY n - Set Y position to n
    • 
                        
  • SETHEADING n or SETH n - Set heading (0=right, 90=up)
    • 
                        
  • HOME - Return to center (0, 0) facing up
    • 
                    

                

                

                    
Pen Control

                    
    
                        
  • PENUP or PU - Lift pen (stop drawing)
    • 
                        
  • PENDOWN or PD - Lower pen (start drawing)
    • 
                        
  • PENSIZE n - Set pen width to n pixels
    • 
                        
  • SETPENCOLOR r g b - Set pen color (RGB 0-255)
    • 
                    

                

                

                    
Display & Shapes

                    
    
                        
  • CLEAR - Clear all drawings
    • 
                        
  • HIDETURTLE or HT - Hide the turtle
    • 
                        
  • SHOWTURTLE or ST - Show the turtle
    • 
                        
  • CIRCLE radius - Draw a circle
    • 
                        
  • BOX width height - Draw a rectangle
    • 
                        
  • SQUARE size - Draw a square
    • 
                    

                

                

                    
Variables & Control

                    
    
                        
  • MAKE "var value - Set variable
    • 
                        
  • :var - Use variable value
    • 
                        
  • REPEAT n [commands] - Repeat commands n times
    • 
                        
  • TO name :param1 :param2 ... [commands] END - Define procedure
    • 
                    

                

                

                    
Arithmetic & Math

                    
    
                        
  • +, -, *, / - Basic arithmetic
    • 
                        
  • MOD - Modulo operation
    • 
                        
  • SQRT n - Square root
    • 
                        
  • SIN n, COS n, TAN n - Trigonometry (degrees)
    • 
                        
  • ABS n - Absolute value
    • 
                        
  • ROUND n, FLOOR n, CEILING n - Rounding
    • 
                        
  • RANDOM n - Random integer 0 to n-1
    • 
                        
  • ( ) - Parentheses for grouping
