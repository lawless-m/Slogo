Available Commands

            

                

                    
Movement

                    
    
                        
  • FORWARD n or FD n - Move forward n pixels
    • 
                        
  • BACKWARD n or BK n - Move backward n pixels
    • 
                        
  • LEFT n or LT n - Turn left n degrees
    • 
                        
  • RIGHT n or RT n - Turn right n degrees
    • 
                        
  • SETXY x y - Move to position (x, y)
    • 
                        
  • SETX n - Set X position to n
    • 
                        
  • SETY n - Set Y position to n
    • 
                        
  • SETHEADING n or SETH n - Set heading (0=right, 90=up)
    • 
                        
  • HOME - Return to center (0, 0) facing up
    • 
                    

                

                

                    
Pen Control

                    
    
                        
  • PENUP or PU - Lift pen (stop drawing)
    • 
                        
  • PENDOWN or PD - Lower pen (start drawing)
    • 
                        
  • PENSIZE n - Set pen width to n pixels
    • 
                        
  • SETPENCOLOR r g b - Set pen color (RGB 0-255)
    • 
                    

                

                

                    
Display & Shapes

                    
    
                        
  • CLEAR - Clear all drawings
    • 
                        
  • HIDETURTLE or HT - Hide the turtle
    • 
                        
  • SHOWTURTLE or ST - Show the turtle
    • 
                        
  • CIRCLE radius - Draw a circle
    • 
                        
  • BOX width height - Draw a rectangle
    • 
                        
  • SQUARE size - Draw a square
    • 
                    

                

                

                    
Variables & Control

                    
    
                        
  • MAKE "var value - Set variable
    • 
                        
  • :var - Use variable value
    • 
                        
  • LOCAL "var or LOCAL [var1 var2] - Declare local variable(s)
    • 
                        
  • REPEAT n [commands] - Repeat commands n times
    • 
                        
  • WHILE condition [commands] - Loop while condition is true
    • 
                        
  • FOR [var start end increment] [commands] - Loop with counter (increment optional)
    • 
                        
  • TO name :param1 :param2 ... [commands] END - Define procedure
    • 
                        
  • OUTPUT value - Return value from procedure
    • 
                        
  • STOP - Exit procedure without returning value
    • 
                        
  • PRINT value or PR value - Output value to console
    • 
                    

                

                

                    
Arithmetic & Math

                    
    
                        
  • +, -, *, / - Basic arithmetic
    • 
                        
  • MOD - Modulo operation
    • 
                        
  • POWER base exp or POW base exp - Exponentiation (base^exp)
    • 
                        
  • SQRT n - Square root
    • 
                        
  • SIN n, COS n, TAN n - Trigonometry (degrees)
    • 
                        
  • ABS n - Absolute value
    • 
                        
  • ROUND n, FLOOR n, CEILING n - Rounding
    • 
                        
  • RANDOM n - Random integer 0 to n-1
    • 
                        
  • ( ) - Parentheses for grouping
    • 
                    

                

                

                    
Comparisons

                    
    
                        
  • < - Less than
    • 
                        
  • > - Greater than
    • 
                        
  • = - Equal to
    • 
                        
  • <= - Less than or equal
    • 
                        
  • >= - Greater than or equal
    • 
                        
  • <> - Not equal
    • 
                    

                

                

                    
Logical Operators

                    
    
                        
  • AND - Logical AND
    • 
                        
  • OR - Logical OR
    • 
                        
  • NOT - Logical NOT
    • 
                    

                

                

                    
Conditionals

                    
    
                        
  • IF condition [commands] - Execute if true
    • 
                        
  • IFELSE condition [true] [false] - Conditional execution
    • 
                    

                

                

                    
Query Functions

                    
    
                        
  • XCOR - Returns current X coordinate
    • 
                        
  • YCOR - Returns current Y coordinate
    • 
                        
  • HEADING - Returns current heading (0-360)
    • 
                        
  • PENDOWN? or PENDOWNP - Returns 1 if pen is down, 0 if up
    • 
                        
  • PENSIZE - Returns current pen size
    • 
                        
  • PENCOLOR - Returns current pen color as list [r g b]
    • 
                        
  • REPCOUNT - Returns current iteration number in REPEAT loop (1-indexed)
    • 
                    

                

                

                    
Lists & List Operations

                    
    
                        
  • [item1 item2 ...] - List literal
    • 
                        
  • FIRST list - Returns first element
    • 
                        
  • LAST list - Returns last element
    • 
                        
  • BUTFIRST list or BF list - All but first element
    • 
                        
  • BUTLAST list or BL list - All but last element
    • 
                        
  • ITEM n list - Get nth element (1-indexed)
    • 
                        
  • COUNT list - Returns list length
    • 
                        
  • EMPTY? list - Returns 1 if empty, 0 otherwise
    • 
                        
  • FPUT item list - Add item to front of list
    • 
                        
  • LPUT item list - Add item to end of list
    • 
                        
  • LIST item1 item2 - Create list from items
    • 
                        
  • SENTENCE list1 list2 or SE list1 list2 - Flatten/concatenate
    • 
                        
  • MEMBER? item list or MEMBERP item list - Check if item in list
    • 
                        
  • POSITION item list - Find 1-based index (0 if not found)
    • 
                    

                

                

                    
Higher-Order Functions

                    
    
                        
  • MAP "proc list - Apply procedure to each element
    • 
                        
  • FILTER "pred list - Keep elements where predicate is true
    • 
                        
  • REDUCE "op list - Reduce list to single value
    • 
                        
  • APPLY "proc arglist - Call procedure with argument list
    •
